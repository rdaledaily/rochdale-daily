Naming examples

manchester_road_rochdale.jpg — used on stories mentioning Manchester Road
hollingworth_lake_littleborough.jpg — stories about Hollingworth Lake
yorkshire_street_rochdale.jpg — stories about Yorkshire Street
rochdale_town_hall.jpg — stories mentioning the Town Hall
falinge_park.jpg — stories about Falinge Park
Use at least two words. The more specific the name, the more precisely it matches. If no place photo matches a story, it falls back to the area photo, then to a generated card.

Then, on GitHub

  1. Browse to assets/img/places/ (create the folder if it isn't there)
  2. Add file → Upload files, drag in the downloaded photo
  3. Commit
  4. If you entered a credit, paste the line into assets/img/places/credits.json
